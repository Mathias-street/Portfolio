Encryption and Decryption

This is project uses the command line to encrypt and decrypt files. There are four different parts to the project. Part one you can choose how you want to encrypt a plaintext file. You can choose the mode of encryption and choose the output file name. Part two you choose a password to encrypt the file with, and then decrypt that file using that password. Part three measured the time to decrypt for a long text and a short text with different modes as well. Lastly part four trys to brute force the password of a file.
